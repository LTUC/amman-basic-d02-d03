# Class-02: VCS, Git, GitHub, Markdown

## Outcomes
Students will:
* Learn what is Git 
* Learn basic Git commands 
* Learn what is Github
* Learn the difference between Git and Github 
* Learn Markdown language with its features
* Clone a repository using SSH/HTTPS
